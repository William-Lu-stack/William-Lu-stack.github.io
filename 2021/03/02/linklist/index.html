Data Structure-To implement LinkedList

In contrast, LinkedList data structure is not based on array unlike ArrayList. It consists of nodes, each of nodes is composed of
data and a pointer which point to the next node (last node’s pointer points to null). It spends much less time than in ArrayList when
we add elements into that list.

Create node

It consists of a pointer and an Object which can store all kinds of data.
Adding four functions can help it easier to get and set Node’s each property.

Create the LinkedList

The LinkedList is consists of several nodes.
To set head, last and its size is to make the code’s functions’ writing more clearer. E in <> is to represent the data type used in this class.

Implement add function

The parameter e is to represent the data we use add function to add into LInkedList, E represents the data type. As we have had the function setData written in Node class so we can easily use that to set the data in the new node. The ‘last’ represents the last node, in this function the new node added in is right add at the end of the list in default. Also the size++ will calculate the size of this list.

Implement remove function

We use particular index to confirm which element we want to remove from the list. However, unlike ArrayList we cannot use subscript to directly find the node correspond to the index. Consider when the index is 0 or after 0 && < size. If the index is between the head and the last, then you need to first change the previous elements’ pointer points to the index corresponding element’s next element and then change the removing element’s pointer points to null. After all these done, you can remove any element
in this LinkedList.

Implement get function

As LinkedList does not have the ablity to use subscript to find element directly, so we have to go through the whole LinkedList from head and stop at the element which we want, then return its data.

Implement replace function

Just like above, go through the LinkedList and stop at the element we want to replace,
replace its data by using setData.

Implement reverse function

This one is aim to reverse all the element in this LinkedList. The idea is to use head’s pointer to point to the element we want to switch its pointer’s direction so that we will not lose the data after we change the pointer. After the traversal we need to reset the head node to last node and last node to head node.

Implement getSize() function

To add this function is because the original property ‘size’ is defined during using add function. However, after other functions processed, the original size may be changed so we need to use getSize to make sure we always get the correct length of the LinkedList.

Test all the function

Terminal

Sucess!

We can find the LInkedList is much different from ArrayList, it can save more time than using ArrayList when we need to store large amount of data in a list. However, as ArrayList is actually an array so we can use subscript to process it much easier than in LinkedList and more safer because if an element in the LInkedList lose a pointer points to it or some problems else then it may cause large amount of data loss as data in the LinkedList are closely connected.
