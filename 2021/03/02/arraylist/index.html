Data Structure-To implement ArrayList

When we create an array, we have to know the length of the array we want first. We can create a huge length of array to ensure we have enough vacancies for the data input but it will occupy too much memory whcih means this is not an efficient way. List helps us solve this problem, it has exactly same number of places as the number of datas we want to push in. This blog I will try to implement how the ‘List’ function works.

Implement an add function

The letter E in the parentheses represents the data type of the data we use add function to put in the array
array.length + 1 means when we use the add function , one more data will be put in this array so the length of array should extends one place.
After define the ‘newArray’ , we will put the data e which is the data we want to put in the array through add function and set its place at the end of newArray.
Then copy every elements in the original array to newArray.
Finally reset the original array’s address to newArray’s address and now the array’s value in the stack become a new pointer which points to the newArray’s pointer points to.
